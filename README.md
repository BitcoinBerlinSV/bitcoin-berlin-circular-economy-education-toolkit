# Bitcoin Berlin Circular Economy Education Toolkit

Open-source Bitcoin education and onboarding resources developed from real-world usage in circular economies in Berlín and La Laguna, El Salvador.

This repository documents practical workflows used to introduce Bitcoin as a payment tool for merchants and users in everyday contexts.

---

## Purpose

This toolkit exists to make Bitcoin onboarding:
- practical
- understandable
- repeatable
- community-driven

The materials here are based on direct field experience and are designed to be reused by other communities building circular economies.

## How to use this toolkit

- Educators and field teams can use the merchant guide during onboarding
- New users can use the user guide to understand everyday Bitcoin usage
- Communities can adapt the materials to their local context

These resources are intentionally lightweight and practical.

---

## Scope

This repository focuses on:
- onboarding
- everyday usage
- real-world adoption

It intentionally avoids:
- speculative content
- advanced technical topics
- theoretical explanations not used in practice
